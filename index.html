Здесь вы узнаете как сделать страницу в html и как получить 10 от Виталика

Команды
  1. Какие команды нужно знать и что они делают?
    1. <h1> — heading (заголовок), самый большой и важный; <h2> чуть меньше, <h3> ещё меньше, <h4> мелкий заголовок, <h5> ещё более мелкий, <h6> наименее важный и маленький.

      Пример использования: 
      <h1>Главный заголовок</h1>
<h2>Раздел</h2>
<h3>Подраздел</h3>


    2. Чтобы сделать текст в HTML, вам надо использовать тег <p>. Вы вписываете свой текст между тегом <p> и </p>.
      Пример: 
      <p>Ваш текст</p>


    3. <title> — задаёт название страницы (вкладки браузера); отображается во вкладке и в результатах поиска. Пример использования: 
      <head>
  <title>Мой сайт</title>
</head>


    4. <a> — это тег для создания ссылок. Буква "a" означает anchor (якорь). Он используется для перехода на другие страницы, сайты или файлы. Сам по себе тег не работает — ему нужен атрибут href. Пример использования: <a href="https://google.com">Перейти в Google</a> Текст между <a> и </a> — это то, на что нажимают.


    5. <img> — это тег для вставки изображения на страницу. У него нет закрывающего тега, потому что он одиночный. Чтобы картинка работала, нужно указать атрибут src (путь к изображению). Также желательно добавить alt — это текст, который показывается, если изображение не загрузилось.

      Пример использования: <img src="Название папки в которой лежит картинка/Название картинки.jpg" alt="Описание картинки">

      src — путь к файлу
      alt — описание изображения

      Так же вы можете изменить размеры картинки с помощью атрибутов width и height. Эти атрибуты вставляются прямо в тег <img> после src и alt.

      - width — задаёт ширину изображения в пикселях

      - height — задаёт высоту изображения в пикселях

      Пример использования: 
      <img src="img/название.png" alt="Описание картинки" width="200" height="150">


    6. Тег <br> используется для переноса строки (line break). Он не имеет закрывающего тега, потому что это одиночный элемент.
      С его помощью можно перейти на новую строку внутри текста.

      Пример использования: <br> Первая строка<br>Вторая строка После <br> текст начнётся с новой строки.


    7. Тег <meta> — даёт браузеру информацию о странице (метаданные) и всегда пишется внутри <head>.

      Кодировка UTF-8 используется, чтобы браузер правильно показывал все символы, включая русские, эмодзи и специальные знаки.

      Пример использования: <head> <meta charset="UTF-8"> </head>


    8. Тег <style> используется для написания CSS прямо в HTML-документе и всегда находится внутри <head>. С его помощью можно задавать стили для текста, кнопок, фонов и всего на странице. Пример использования:

      <head>
      <style> body { background-color: lightblue; }

      h1 { color: darkblue; }
      </style>
      </head>
  2. Конкретнее про style

    Тег <style> позволяет делать страницу красивее — менять цвета, шрифты, размеры и другие визуальные элементы прямо в HTML.

    Тег <style> пишется внутри <head>. После него указываешь, что хочешь стилизовать (например p), потом фигурные скобки { } и внутри ставишь команды. Пример: <style> p { color: red; } <-- одна из команд </style> Внутри <style></style> можно вместить сколько угодно таких команд и сколько угодно элементов, которые хочешь сделать красивыми. Не нужно создавать много тегов <style> — одного достаточно для всей страницы.


    Примеры нужных команд (простых)

    1. Команда font-size задаёт размер текста. Правильно пишется внутри фигурных скобок, после названия элемента: Пример: p { font-size: 50px; }

      Размер указывается в пикселях (px). Можно менять любое число, чтобы текст стал больше или меньше.


    2. Команда color задаёт цвет текста. Правильно пишется внутри фигурных скобок, после названия элемента:

      Пример: p { color: red; }

      Вместо red можно использовать любой цвет: название (blue), HEX (#FF0000) или RGB (rgb(255,0,0)).


    3. Команда background-color задаёт цвет фона элемента.
      Правильно пишется внутри фигурных скобок, после названия элемента:

      Пример: p { background-color: yellow; }

      Вместо yellow можно использовать название цвета, HEX (#FFFF00) или RGB (rgb(255,255,0)).
Куда что вставлять?
  1. head

    1. <title> — задаёт название страницы (вкладка браузера)


    2. <meta charset="UTF-8"> — указывает кодировку страницы, чтобы отображались все символы, включая русские


    3. <style> — сюда пишем CSS, чтобы задавать стили для текста, фонов, кнопок и других элементов
  2. body

    1. <h1> — заголовки, от <h1> (самый большой) до <h6> (самый маленький)


    2. <p> — абзацы текста


    3. <a href="..."> — ссылки на другие страницы или сайты


    4. <img src="..." alt="..."> — вставка изображений


    5. <br> — перенос строки

Для начала нужно создать отдельную папку в которой вы будете работать. После этого, вы переходите в visual studio code и выбираете её. Далее вам нужно создать внутри этой папки ещё папки для css и img. Для этого вам нужно нажать сюда.

После создания папок вам нужно создать сам html документ который вы должны назвать "любое слово.html". Для этого вам нужно нажимать сюда.

Так же для того что бы было удобнее пользоваться программой, вы можете установить плагин live preview. Для этого вам нужно зайти сюда и вбить в поиске название которое было указано выше. После установки вы можете нажать на свой html файл правой кнопкой мыши и выбрать show preview.
